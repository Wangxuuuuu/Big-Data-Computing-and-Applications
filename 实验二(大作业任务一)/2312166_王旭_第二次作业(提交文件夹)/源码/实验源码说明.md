# 推荐系统实验源码说明

## 运行环境

- Python 3.8.0
- 推荐模型、调参与测试代码仅使用 Python 标准库，无需安装第三方依赖。

## 入口脚本

- `run_baselines.py`：在固定验证集上运行全局均值、偏置基线、Item-CF 和矩阵分解，并统计 RMSE、训练时间、预测时间与峰值内存。
- `tune_matrix_factorization.py`：比较矩阵分解候选参数。
- `tune_ensemble.py`：计算 Item-CF 与矩阵分解的融合权重。
- `generate_predictions.py`：在完整训练集上训练最终融合模型，生成并校验 `output/final/result.txt`。

## 核心模块

`src/` 包含数据读取与划分、评价指标、全局均值与偏置基线、Item-CF、矩阵分解、模型融合和统一参数配置。`tests/test_experiment.py` 包含 6 项自动化测试，覆盖数据解析、可复现划分、RMSE、冷启动处理及主要模型行为。

## 运行方法

运行时应保持脚本、`src/`、`tests/`、`data/` 和 `output/` 位于同一项目根目录。在项目根目录执行：

```bash
python run_baselines.py
python tune_matrix_factorization.py
python tune_ensemble.py
python generate_predictions.py
python -m unittest discover -s tests -v
```

其中，`run_baselines.py` 会在固定验证集上运行四个单模型；受 Item-CF 相似度计算影响，完整对比实验约需两分钟，该时间不属于正式测试集预测耗时。

`generate_predictions.py` 在完整训练集上训练最终融合模型并预测正式测试集，通常约需 30 秒至 1 分钟。正式测试集不含真实评分标签，报告中的 RMSE 17.1500 是固定验证集结果，不是正式测试集成绩。
